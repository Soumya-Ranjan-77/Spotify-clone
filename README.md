# 🎵 Spotify Clone

A simple **Spotify Clone** built using **HTML** and **CSS**. This project is designed to replicate the basic user interface of Spotify while improving front-end development skills. It focuses on creating a responsive and visually appealing music streaming platform layout without using JavaScript or any backend technologies.

## 🚀 Features

* Modern Spotify-inspired user interface
* Responsive design for different screen sizes
* Navigation sidebar
* Playlist section
* Music cards with album artwork
* Footer music player design
* Clean and organized code structure
* Beginner-friendly project

## 🛠️ Technologies Used

* HTML5
* CSS3

This project was created for learning and practicing front-end web development concepts. It helps in understanding:

* HTML page structure

* Flexbox and Grid layouts
* Responsive web design
* UI cloning and design implementation
